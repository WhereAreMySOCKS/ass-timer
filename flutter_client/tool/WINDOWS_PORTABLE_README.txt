Ass-Timer 2.0.0 (Windows x64 portable)

1. Extract the entire ZIP to a writable folder.
2. Run Ass-Timer.exe. Keep the data and DLL files beside the executable.
3. Use the system tray icon to show/hide the pet, open the control center, or exit.
4. Updates open in your browser and never install automatically.

Diagnostics:
- Logs: %LOCALAPPDATA%\AssTimer\logs
- Native crash dumps: %LOCALAPPDATA%\AssTimer\crashes
- The same log folder can be opened from Settings > About > Open logs folder.
- If the tray icon cannot be created, the pet remains available from the taskbar.
- If startup fails, keep the ZIP fully extracted and send the newest log and .dmp file.

Supported systems: Windows 10/11 x64.
